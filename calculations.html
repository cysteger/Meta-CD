How These Calculations Work

This page summarizes the mathematical framework used by Meta‑CD to compute species‑specific coverage, required sequencing depth, minimum detectable abundance, and MAG recovery potential.

Core Coverage Model

All Meta‑CD outputs derive from the standard metagenomic coverage equation:

C = Dspecies / G

Where:

  • G = genome size (Mb)
  • Dspecies = bases sequenced from the species

Because shotgun sequencing distributes reads in proportion to relative abundance (A):

Dspecies = Deffective × 1000 × A

DNA‑Limited Effective Depth

Meta‑CD incorporates DNA‑input constraints using:

Deffective = min(Dcommunity, Dmax)

Where:

Dmax = DNAng

This reflects the empirical approximation that 1 ng DNA ≈ 1 Gbp unique sequence. If sequencing depth exceeds this limit, coverage becomes DNA‑limited.

Required Sequencing Depth

The sequencing depth needed to reach a target coverage is obtained by solving the coverage equation for community depth:

Drequired = (Ctarget × G) / (1000 × A)

This value is DNA‑adjusted using the effective depth defined in the Core Coverage Model.

Minimum Relative Abundance Required

The minimum abundance needed to reach a target coverage is:

Amin = (Ctarget × G) / (1000 × Deffective)

Lower sequencing depth or larger genomes increase the required abundance.

Bases Sequenced for This Species

The expected number of bases originating from a species is:

Bspecies = Dcommunity × 109 × A

This determines how deeply a species is represented in the dataset.

Coverage From Sequencing Depth

Substituting abundance‑scaled depth into the coverage equation yields:

C = (Deffective × 1000 × A) / G

Coverage increases with sequencing depth and abundance, and decreases with genome size.

MAG Recovery Potential

MAG recovery likelihood is interpreted using coverage thresholds:

  • >20× — high likelihood
  • 10–20× — moderate
  • 1–5× — low
  • <1× — very unlikely

These thresholds reflect typical assembly performance across metagenomic datasets.

Coverage Estimation Table

The table applies the same coverage formula across a grid of sequencing depths and abundances, incorporating DNA‑limited depth where applicable. It provides a rapid reference for evaluating feasibility of detection, profiling, and assembly.
